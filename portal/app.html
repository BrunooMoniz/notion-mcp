🔌 Conectar ao seu assistente (Claude, Cursor…)

Seu Zinom vira uma ferramenta no seu assistente. Depois é só pedir "busca no meu Zinom: …".

Endereço MCP: —

No Claude.ai / Claude Desktop: Configurações → Conectores → Adicionar conector personalizado → cole o endereço MCP acima. Ele vai pedir seu e-mail e um código de 6 dígitos para autorizar (não precisa do token abaixo). No Claude Code / Cursor: gere o token abaixo e use o comando.

⚠️ Copie agora: o token aparece uma única vez. Se perder, é só gerar outro (o anterior deixa de valer).

Outros clientes MCP (Cursor, etc.): use o endereço acima com o cabeçalho Authorization: Bearer <seu token>.

Conectar com um token (mais simples) — tutorial
  1. Abra notion.so/my-integrations → New integration (tipo Internal). Dê o nome Zinom e salve. Copie o Internal Integration Secret (começa com ntn_).
  2. Importante: abra a página (ou a raiz do seu workspace) que você quer no Zinom, clique no menu ••• (canto superior direito) → Connections / Conexões → adicione a integração Zinom. Sem esse passo, o Zinom não enxerga nem indexa nada.
  3. Cole o token aqui e conecte:

O token é mais fácil de configurar que o OAuth (botão acima) e fica guardado cifrado. Você compartilha só as páginas que quiser conectando-as à integração.
